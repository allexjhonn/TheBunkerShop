Este é o projeto do meu grupo no curso técnico de t.i.
Utilizando nossos conhecimentos adquiridos no curso, construimos linha por linha desse site que usamos de tema uma loja que fica em um bunker pós-apocaliptico e vendemos itens para sobrevivência.
As linguagens utilizadas foram HTML, CSS e JavaScript. Com HTML criamos a base do site, com essa linguagem adicionamos o conteúdo do site.
Com o CSS estilizamos a página e deixamos visualmente linda adicionando itens, imagens e objetos de acordo com o tema.
O JAVASCRIPT foi útil pra nós, pois demos vida ao nosso site incluindo ações a botões, animações e até ações que acontecem por trás do site, onde nem o usuário vê.
E acho que o JAVASCRIPT foi a chave pra conseguirmos finalizar o projeto com chave de ouro.
